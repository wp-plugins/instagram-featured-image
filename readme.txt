== Installation ==

### Installation

1. Upload 'instagram-featured-image' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Appearance > Widgets > Instagram Featured Image.
4. Drag and drop "Instagram Featured Image" widget and open it and enter instagram image link.
5. Enjoy.
